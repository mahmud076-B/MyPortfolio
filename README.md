## üì¶ Installation


2. Install dependencies:

3. Start the development server:

## üé® Customization

### Update Your Information

1. **Projects**: Edit `src/data/projects.js`
2. **Skills**: Edit `src/data/skills.js`
3. **Experience**: Edit `src/data/experience.js`
4. **Social Links**: Update links in `src/components/layout/Footer.jsx` and `src/components/sections/Contact.jsx`

### Colors

Customize colors in `tailwind.config.js`:
- `bg-primary`: Primary background
- `bg-secondary`: Secondary background
- `accent-blue`: Blue accent
- `accent-purple`: Purple accent
- `accent-cyan`: Cyan accent
